# Code of Conduct

We are committed to fostering an inclusive, respectful, and professional environment for everyone. All participants are expected to:

- **Respect others:** Treat all community members with courtesy and consideration.
- **Be constructive:** Provide helpful feedback and refrain from personal attacks.
- **Avoid harassment:** Do not engage in harassment, hate speech, or any behavior that undermines the well-being of others.
- **Report issues:** If you experience or witness any behavior that violates this Code, please report it to the project maintainers.

Violations of this Code of Conduct may result in temporary or permanent exclusion from the project community. We appreciate your help in keeping our community welcoming and safe.

*This Code of Conduct is subject to updates. Please check back periodically for any changes.*
